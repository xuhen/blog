二叉树的遍历（前序，中序，后序）
发布于：2020-04-26 10:51:57 标签：/ algorithm / 访问：

二叉树，这样的结构还是很常见的，二叉树的遍历更是数型算法的基础。那今天就介绍下二次数的非递归遍历。二叉树分前序遍历、中序遍历、后序遍历。

如果按照下面的树：
前序： 根 -> 左 -> 右 => 1, 2, 4, 5, 3, 6, 7
中序： 左 -> 根 -> 右 => 4, 2, 5, 1, 6, 3, 7
后序： 左 -> 右 -> 根 => 4, 5, 2, 6, 7, 3, 1

二叉树的三种遍历都要使用stack（栈）这样的数据结构。
